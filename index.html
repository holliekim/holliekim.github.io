About

Current Work:

Hollie is working as a GIS Technician at the Minnesota Geospatial Information office (State of Minnesota, MN IT Services), and has been in this position since 2015. She is putting her skills to work on the Next Generation 9-1-1 (NG9-1-1) GIS project. As a Regional NG9-1-1 GIS Representative, her responsibilities include geospatial data analysis, comparing spatial data attributes to 911 system data, report review, communicating results to county GIS coordinators, and providing feedback and guidance. Other project tasks include map design and development (static and interactive), maintaining project status layers on ArcGIS Online, translating data into state standards using FME, and leveraging python to automate tasks as necessary.


Goals:

Continue to develop skills in JavaScript, and GIS focused web developement, expand on current Python knowledge, and become more familiar with PostgreSQL/PostGIS.


Past Work Experience:
